## 📖 Sobre

Este é meu **primeiro projeto de HTML e CSS** — uma tela de login estática desenvolvida como exercício introdutório durante meus primeiros contatos com desenvolvimento web.

> ⚠️ **Aviso:** Este projeto é propositalmente básico. Não tem JavaScript, não tem backend, não tem validação — é puro HTML e CSS. Está aqui como registro do ponto de partida, não como referência de boas práticas.

O projeto foi feito no contexto da **PotiCode**, Empresa Júnior da UFRN, uma iniciativa de introdução à programação, e representa os primeiros passos no front-end.


---

## 🖥️ Como visualizar

Não precisa instalar nada. Basta clonar e abrir o arquivo no navegador:

```bash
git clone https://github.com/peddrela/PotiCode-Login.git
cd PotiCode-Login

# Abra o arquivo diretamente no navegador
xdg-open index.html        # Linux
open index.html            # macOS
start index.html           # Windows
```
